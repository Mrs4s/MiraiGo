# MiraiGo
qq-android协议的golang实现 移植于mirai

# 警告
本项目为协议实现，api非常原始，并不推荐使用。

建议基于 [go-cqhttp](https://github.com/Mrs4s/go-cqhttp) 使用框架开发。

同时提供不基于 CQHTTP 的原生框架 [MiraiGo-Template](https://github.com/Logiase/MiraiGo-Template) 进行开发。
